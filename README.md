# 泰科力合销售CRM管理系统

## 使用

安装项目依赖
```
npm install
```

开发

```
npm run dev
```

打包
```
npm run build
```

## 项目目录

- build - 打包文件
- config - 配置
- dist - 最终上线代码
- node_modules - 包
- src - 主要代码
    - assets - 公共插件
    - components - 组件
    - router - 路由
    - APP.vue - 根组件
    - main.js - 主要文件
- static - 静态资源
- index.html - 入口文件
- package.json - 依赖包配置文件
